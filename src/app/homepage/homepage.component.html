Gaijin Explorer is a project to primarily serve as an excuse to learn new technologies and frameworks. The idea being to create a simple manga reading application using a given technology/framework. Thankfully this is made possible to Manga Eden and their free API (BTW go support them they are awesome!)

All of the Gaijin Explorer projects are publically availible and open-sourced under the MIT licence. This means that you can download the download the code to learn from or use in your own project. You can find all of the Gaijin Explorer projects at my Github.

So go have fun and explore the site. I will periodically update this website as well as my blog. If you encounter bugs or have a new feature request for any of the projects, use Github's issue system (or compain about it in the comments section cuz you're lazy, cuz I totally read people's comments).
